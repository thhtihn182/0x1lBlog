# Giới thiệu

Đang trong quá trình phát triển
Một hệ thống blog sử dụng Spring Boot và Vue, tách biệt quá trình phát triển giao diện người dùng (front-end) và máy chủ (back-end) nhưng vẫn giữ kết nối giữa người dùng.

Đây là một blog cá nhân, được duy trì lâu dài

## Backend

1. Framework chính：[Spring Boot](https://github.com/spring-projects/spring-boot)
2. Framework bảo mật：[Spring Security](https://github.com/spring-projects/spring-security)
3. Token：[jjwt](https://github.com/jwtk/jjwt)
4. ORM Framework：[MyBatis](https://github.com/mybatis/spring-boot-starter)
5. Plugin phân trang：[PageHelper](https://github.com/pagehelper/Mybatis-PageHelper)
6. Markdown 转 HTML：[commonmark-java](https://github.com/commonmark/commonmark-java)

...............vv


## Frontend

Framework chính：Vue3.x、Vue Router、Pinia

Dự án Vue được xây dựng dựa trên  Vite + Vue 3.

Thư viện JS và CSS tham khảo：[axios](https://github.com/axios/axios)、[dayjs](https://github.com/iamkun/dayjs)、[nprogress](https://github.com/rstacruz/nprogress)、[prismjs](https://github.com/PrismJS/prism)、 [lodash-es](https://github.com/lodash/lodash)、[mavonEditor](https://github.com/hinesboy/mavonEditor)
